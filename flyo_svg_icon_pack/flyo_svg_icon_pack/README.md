# Flyo SVG Icon Pack

This pack contains reusable outline SVG icons for the Flyo travel website.

## Key points
- 24x24 SVG icons
- `stroke="currentColor"` so icons inherit CSS color
- Good for current UI and future expansion
- Folders grouped by use case

## Folders
- `current-use/trust-strip/`
- `current-use/package-meta/`
- `future-use/filters/`
- `future-use/travel-styles/`
- `future-use/general/`

## Usage example
```html
<img src="./current-use/trust-strip/visa-assistance.svg" alt="Visa assistance" />
```

Or inline for color control:
```css
.icon { width: 20px; height: 20px; color: #102694; }
```

## Notes
- Current icons cover trust strip, package meta, WhatsApp, filters, and travel styles.
- Future-use icons cover common travel/package UI needs like breakfast, hotel, airport transfer, and support.
